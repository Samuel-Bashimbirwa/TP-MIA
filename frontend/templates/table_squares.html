HTML Tags Explanation

<table>

Defines the structure of the data table. It acts as the parent container wrapping all rows, columns, and headings.

<tr>

Stands for Table Row. It defines a horizontal row of cells in the table.

<th>

Stands for Table Header. Used inside a <tr> row to define a header cell, typically rendered in bold and centered by default.

<td>

Stands for Table Data. Used to define standard cells that hold text, numbers, or interactive elements inside a table row.
